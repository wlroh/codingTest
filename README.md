## Coding Test Practice

 - ### Commit Type
   - add: 신규 문제 추가에 대한 커밋
   - fix: 문제 풀이 테스트 실패관련 수정에 대한 커밋
   - test: 테스트 케이스 추가관련 커밋
   - refactor: 코드 리팩토링에 대한 커밋
   - docs: 문서 수정에 대한 커밋
